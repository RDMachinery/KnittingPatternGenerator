╔══════════════════════════════════════════════════╗
║  PATTERN 2: FEATHER AND FAN LACE                 ║
╚══════════════════════════════════════════════════╝

A delicate, undulating lace pattern. Beautiful for shawls and scarves.
Cast on: multiple of 18 stitches
Use a smooth, light-coloured yarn to show the lace to best effect.

KEY:
  . = Knit stitch
  - = Purl stitch
  O = Yarn over (creates lace hole)
  / = K2tog (knit 2 together, right-leaning decrease)
  \ = SSK (slip slip knit, left-leaning decrease)

    +------------------------------------------------------+
  1 │------------------------------------------------------│  ← RS
  2 │///\O.O.O.O\///...///\O.O.O.O\///...///\O.O.O.O\///...│  → WS
  3 │------------------------------------------------------│  ← RS
  4 │......................................................│  → WS
  5 │------------------------------------------------------│  ← RS
  6 │///\O.O.O.O\///...///\O.O.O.O\///...///\O.O.O.O\///...│  → WS
  7 │------------------------------------------------------│  ← RS
  8 │......................................................│  → WS
  9 │------------------------------------------------------│  ← RS
 10 │///\O.O.O.O\///...///\O.O.O.O\///...///\O.O.O.O\///...│  → WS
 11 │------------------------------------------------------│  ← RS
 12 │......................................................│  → WS
 13 │------------------------------------------------------│  ← RS
 14 │///\O.O.O.O\///...///\O.O.O.O\///...///\O.O.O.O\///...│  → WS
 15 │------------------------------------------------------│  ← RS
 16 │......................................................│  → WS
    +------------------------------------------------------+

WRITTEN INSTRUCTIONS:
Row 1 (RS): Knit all stitches.
Row 2 (WS): Purl all stitches.
Row 3 (RS): *K2tog x3, (yo, k1) x6, k2tog x3* repeat to end.
Row 4 (WS): Purl all stitches.
Repeat rows 1-4 for pattern.

Note: Row 3 is stitch-count neutral — each decrease is offset by a yarn over.
